# Pilot Onboarding Checklist — Obra Sys Brasil PME

## Antes Do Onboarding

- [ ] Confirmar empresa selecionada para piloto controlado.
- [ ] Registrar responsavel principal da empresa.
- [ ] Confirmar aceite de dados limitados.
- [ ] Confirmar aceite de feedback frequente.
- [ ] Confirmar que a empresa nao exige funcionalidades fora do escopo.
- [ ] Confirmar backup/snapshot antes de inserir dados reais.
- [ ] Confirmar ambiente correto.
- [ ] Confirmar que RLS, Storage e Edge Functions seguem aprovados.

## Criacao Da Empresa

- [ ] Criar organizacao.
- [ ] Criar usuario admin.
- [ ] Criar usuario operacional, se necessario.
- [ ] Confirmar membership e papel de cada usuario.
- [ ] Confirmar que usuario sem permissao financeira nao ve custo, margem ou lucro.
- [ ] Configurar dados basicos da empresa.

## Primeiro Fluxo Guiado

- [ ] Criar primeiro orcamento PME com o cliente.
- [ ] Adicionar ambiente.
- [ ] Adicionar item manual.
- [ ] Criar primeiro item do Meu Catalogo.
- [ ] Gerar primeira proposta.
- [ ] Validar que proposta cliente nao mostra custo, margem ou lucro.
- [ ] Aprovar ou marcar orcamento conforme fluxo combinado.
- [ ] Converter orcamento em obra teste.
- [ ] Criar primeira tarefa.
- [ ] Criar primeiro custo.
- [ ] Criar primeiro recebimento.
- [ ] Criar primeiro diario.
- [ ] Adicionar primeira foto.
- [ ] Gerar primeiro relatorio cliente.
- [ ] Validar que relatorio cliente nao mostra custo, margem ou lucro.

## Treinamento Minimo

- [ ] Explicar que Axia e consultiva e nao altera dados sozinha.
- [ ] Explicar diferenca entre proposta cliente e visoes internas.
- [ ] Explicar como reportar erro.
- [ ] Explicar limites do piloto.
- [ ] Explicar que dados criticos devem ser conferidos manualmente.
- [ ] Confirmar que usuario entendeu o fluxo principal.

## Encerramento Do Onboarding

- [ ] Registrar data do onboarding.
- [ ] Registrar usuarios criados.
- [ ] Registrar obras/orcamentos iniciais.
- [ ] Registrar duvidas levantadas.
- [ ] Registrar riscos percebidos.
- [ ] Agendar check-in do Dia 1.
